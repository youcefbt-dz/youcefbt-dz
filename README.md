## MarketLab

> *An institutional-grade open-source quantitative finance research platform.*

MarketLab started as a single Python script for portfolio analysis. It is now a fully deployed Flask web application with:

| Module | Description |
|---|---|
| **Analyze** | 10-indicator technical dashboard ¬Ј Bollinger Bands ¬Ј Monte Carlo GBM simulation ¬Ј AI signal rationale via Gemini ¬Ј Sentiment meter ¬Ј Peer comparison |
| **Backtest** | v4.0 engine ¬Ј Short selling ¬Ј Kelly criterion sizing ¬Ј Portfolio drawdown kill-switch ¬Ј Sortino / Calmar / Omega / VaR / CVaR |
| **Optimizer** | Bayesian parameter tuning via Optuna across 26 signal parameters |
| **ML Predictor** | Ensemble: Random Forest + Logistic Regression + XGBoost ¬Ј SMOTE ¬Ј SHAP explainability ¬Ј Drift detection |
| **Sentiment** | VADER ¬Ј Financial keyword boosting ¬Ј Time-weighted scoring ¬Ј Tail risk detection |
| **ML Chain** | Ethereum Sepolia testnet wallet ¬Ј MetaMask integration ¬Ј Etherscan API |
| **Scanner / Warehouse** | Multi-asset screener ¬Ј Persistent reliability scoring via black-box logger |

**Stack:** Python ¬Ј Flask ¬Ј Jinja2 ¬Ј Chart.js ¬Ј Pandas ¬Ј NumPy ¬Ј Scikit-learn ¬Ј XGBoost ¬Ј Optuna ¬Ј VADER ¬Ј Gemini API ¬Ј Web3.js ¬Ј Leaflet ¬Ј Render
