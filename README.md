## ◈ Projects

<br/>

**`▸ MarketLab`** &nbsp;—&nbsp; Quant Finance Research Platform &nbsp;·&nbsp; [![Live](https://img.shields.io/badge/LIVE-FF2D55?style=flat-square&labelColor=0D1117)](https://app.marketlab.io) [![GitHub](https://img.shields.io/badge/GitHub-0D1117?style=flat-square&logo=github)](https://github.com/youcefbt-dz/MarketLab)

<img align="right" src="https://media0.giphy.com/media/SUcnmocyRKkfSil9xd/giphy.gif" width="260px" />

Open-source platform built from scratch for quantitative finance research. Walk-forward backtesting with no lookahead bias, Monte Carlo simulation across 1,000 permutations, XGBoost + SHAP explainability, Optuna Bayesian optimization, Hidden Markov Model regime detection, and a full metrics suite — Sharpe, Sortino, Calmar, VaR, CVaR, Omega.

`Python` `Flask` `React` `TypeScript` `PostgreSQL` `Supabase` `XGBoost` `hmmlearn` `Optuna`

<br clear="right"/>

**`▸ Chronicle Tracker`** &nbsp;—&nbsp; Manhwa & Manga Reading Tracker &nbsp;·&nbsp; [![GitHub](https://img.shields.io/badge/GitHub-0D1117?style=flat-square&logo=github)](https://github.com/youcefbt-dz)

<img align="right" src="https://media3.giphy.com/media/9zExs2Q2h1EHfE4P6G/giphy.gif" width="220px" />

Personal reading tracker built because existing tools weren't good enough. Tracks series, chapter progress, reading status and backlog — all in a sharp, minimal UI with zero border-radius and a blue-and-white Valorant-inspired design.

`Python` `Flask` `SQLite` `JavaScript` `HTML/CSS`

**`▸ ouchfix`** &nbsp;—&nbsp; AI Terminal Error Assistant &nbsp;·&nbsp; [![GitHub](https://img.shields.io/badge/GitHub-0D1117?style=flat-square&logo=github)](https://github.com/youcefbt-dz)

<img align="right" src="https://media1.giphy.com/media/NfkMAioqnkf7q4LBqA/giphy.gif" width="180px" />

CLI tool that intercepts terminal errors and fixes them instantly with AI. Paste any broken command or error trace — get a corrected command and explanation. Supports **Groq**, **Ollama**, and **Gemini** — works fully offline or in the cloud.

`Python` `Bash` `Groq API` `Ollama` `Gemini API`
